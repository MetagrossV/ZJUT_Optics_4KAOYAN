# 绘图题重设计文档

## 核心需求

用户需要从"手绘线条"模式转变为"交互式光学编辑器"模式：
1. 从列表中选择光学器件（凸透镜、凹透镜、平面镜、物体）
2. 点击画布放置器件
3. 放置后可以拖动调整位置
4. 可以设置器件参数（焦距、折射率等）
5. 添加辅助线（法线、光轴）
6. 在物体上创建光线

## 数据模型

### 器件（OpticalElement）
```javascript
{
  id: string,          // 唯一标识
  type: string,          // 'convex-lens' | 'concave-lens' | 'plane-mirror' | 'object' | 'prism'
  x: number,            // 物理x坐标（相对画布中心）
  y: number,            // 物理y坐标（相对画布中心）
  params: {
    focalLength: number,    // 焦距（透镜）
    aperture: number,       // 孔径（透镜）
    angle: number,          // 倾斜角（平面镜）
    height: number,         // 高度（物体）
    refractiveIndex: number, // 折射率（棱镜）
    // ... 其他参数
  }
}
```

### 光线（Ray）
```javascript
{
  id: string,
  startElementId: string,  // 起始器件（物体）
  endElementId: string,    // 目标器件（透镜/镜子）
  type: 'parallel' | 'through-center' | 'through-focus', // 光线类型
  color: string
}
```

### 画布状态
```javascript
{
  elements: [],      // 器件列表
  rays: [],          // 光线列表
  selectedTool: null, // 当前选择的工具
  selectedElement: null, // 当前选中的器件
  isPlacing: false,  // 是否正在放置
  isDragging: false, // 是否正在拖动
}
```

## 交互流程

### 1. 选择工具
- 底部工具箱显示可用器件列表
- 点击工具箱中的器件，进入"放置模式"
- 光标变为十字，提示点击画布放置

### 2. 放置器件
- 点击画布任意位置，器件放置在该位置
- 放置后自动选中该器件，显示参数面板
- 参数面板可以调整焦距、高度等

### 3. 拖动器件
- 点击已放置的器件，选中它
- 拖动时器件跟随手指移动
- 释放后固定位置

### 4. 创建光线
- 选中物体器件
- 点击"创建光线"按钮
- 选择光线类型（平行光、过中心、过焦点）
- 系统自动计算光线经过其他器件后的路径
- 显示光线和延长线（虚线）

### 5. 添加辅助线
- 选中透镜/镜子
- 点击"添加辅助线"
- 显示法线（虚线，红色）
- 显示焦点位置（F和F'）

### 6. 验证
- 点击"验证"按钮
- 系统检查：
  - 物体位置是否正确
  - 光线是否经过正确路径
  - 像的位置是否正确
- 给出评分和提示

## 技术实现

### 坐标系统
- 物理坐标：原点画布中心，x向右，y向上
- Canvas坐标：原点左上角，x向右，y向下
- 使用 `toCanvas(physX, physY)` 转换

### 绘制顺序
1. 清空画布
2. 绘制网格和光轴
3. 绘制所有器件（按z-order）
4. 绘制所有光线
5. 绘制选中器件的高亮框
6. 绘制放置预览（如果正在放置）

### 触摸事件处理
- `touchstart`: 检测点击位置（工具箱/画布/器件）
- `touchmove`: 拖动器件或绘制预览
- `touchend`: 完成放置或拖动

## 器件绘制函数

每个器件类型有独立的绘制函数，使用 `toCanvas` 转换坐标。

### 凸透镜
- 双凸弧线
- 中心竖线
- 焦距标注

### 凹透镜
- 双凹弧线
- 中心竖线
- 焦距标注

### 平面镜
- 直线段
- 背面短斜线
- 反射面标记

### 物体
- 竖线 + 箭头
- 标注 A/B

## 光线计算

### 凸透镜光线
1. 平行入射 → 过焦点
2. 过中心 → 直线
3. 过焦点 → 平行出射

### 凹透镜光线
1. 平行入射 → 发散（反向延长线过焦点）
2. 过中心 → 直线
3. 指向焦点 → 平行出射

### 平面镜光线
1. 入射角 = 反射角
2. 使用法线计算

## 验证算法

1. 检查物体位置（相对透镜的距离）
2. 检查光线是否正确经过透镜中心或焦点
3. 检查像的位置（使用透镜公式计算）
4. 给出评分：
   - 器件位置正确：30分
   - 光线正确：40分
   - 像位置正确：30分
