## 如何运行

### 1. 打开微信开发者工具
- 下载地址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
- 安装后登录微信账号

### 2. 导入项目
- 点击「导入项目」
- 选择 `E:/Kimi_Project/浙工大光学/mini-program` 目录
- AppID 选择「测试号」（或填写你的小程序 AppID）
- 点击「确定」

### 3. 编译预览
- 等待编译完成
- 在模拟器或真机上预览
- 如遇报错，检查数据文件路径是否正确

### 4. 真机调试
- 点击「真机调试」→「编译并自动调试」
- 使用微信扫描二维码，在手机上体验

## 核心功能说明

### 1. 公式手册（首页）
- **思维导图**：几何光学 / 物理光学，3级展开，点击直达公式/知识点
- **搜索**：支持公式名称、关键词搜索
- **快捷入口**：🔥高频公式（10个）、⚖️易混淆对比（6组）、🃏记忆卡片
- **公式详情**：LaTeX 渲染、变量说明、关联知识点、收藏

### 2. 刷题（选择题）
- **分类筛选**：按章节（几何/物理）和难度（简单/中等/困难）筛选
- **答题模式**：选择答案 → 即时判断 → 答错弹出记忆卡
- **错误诊断**：区分「概念错误」「公式混淆」「符号规则」
- **收藏**：收藏题目到错题本
- **计算题入口**：切换标签进入计算题步骤填空

### 3. 计算题（步骤填空）
- **步骤拆解**：每题拆分为3-4个步骤
- **即时验证**：数值型（容差检查）、公式型（模式匹配）、选择型（精确匹配）
- **错误诊断**：
  - 符号错误：「物距l在物方应为负值」
  - 公式混淆：「你使用了高斯公式，但应先用单球面折射公式」
  - 连续错误：自动推送辅助材料（提示/公式参考/微课）
- **进度条**：步骤进度可视化

### 4. 错题本
- **统计概览**：总答题、正确、错误、正确率
- **薄弱点分析**：按错误类型统计，可视化进度条
- **章节进度**：各章节答题正确率
- **错题列表**：再次练习、查看知识点

### 5. 记忆卡片
- **翻转卡片**：正面知识点名称 → 背面内容+公式
- **填空记忆**：遮住公式关键部分，用户填写验证
- **收藏**：收藏公式到个人收藏夹

## 技术栈

- **前端**：微信小程序原生框架（WXML + WXSS + JS）
- **数据**：本地 JSON（require加载）+ 微信本地存储（wx.getStorageSync）
- **公式渲染**：文本显示（LaTeX字符串），建议后续接入 KaTeX 小程序适配版
- **状态管理**：App.globalData + 本地存储持久化
